# Blog API

### About
This is a generic RESTful API for Blogs made using **Laravel 5.7** and **Eloquent: API Resources**

Laravel 5.7

---
## Users

`POST`
```sh
http://localhost:8000/api/v1/users
```

`GET`
```sh
http://localhost:8000/api/v1/users/{user_id}
```

`PUT`
```sh
http://localhost:8000/api/v1/users/{user_id}
```

`DELETE`
```sh
http://localhost:8000/api/v1/users/{user_id}
```
